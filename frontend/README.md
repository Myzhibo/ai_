# FIRE Advisor iOS App

## 项目创建步骤

### 1. 打开 Xcode
- 打开 Xcode
- 选择 "Create a new Xcode project"
- 选择 "iOS" → "App"

### 2. 项目配置
```
Product Name: FIREAdvisor
Team: [选择你的开发团队]
Organization Identifier: com.fire.advisor
Interface: SwiftUI
Language: Swift
```

### 3. 保存位置
选择 `/Users/doubo/Desktop/gate/ai_/frontend` 作为项目位置

### 4. 依赖管理 - 使用 Swift Package Manager

在 Xcode 中添加以下包:
- File → Add Package Dependencies

#### 添加 Alamofire (网络请求)
```
https://github.com/Alamofire/Alamofire.git
Version: 5.8.0 或更高
```

#### 添加 KeychainSwift (安全存储)
```
https://github.com/evgenyneu/keychain-swift.git
Version: 20.0.0 或更高
```
