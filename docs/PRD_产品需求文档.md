## 1. 产品定位

为年轻人打造的FIRE(财务自由)生活方式平台，集成养老规划、社区社交、城市发现、AI理财顾问于一体。

**核心价值主张**: 为年轻规划，为自由而生

**目标用户**:
- 25-40岁追求财务自由的年轻人
- 关注养老规划的职场人士
- 数字游民和远程工作者
- 对单身经济和生活方式有探索欲的群体

**产品愿景**:
让年轻人提前规划财务自由，探索更多元的生活方式选择

---

## 2. 核心功能模块

### M1: 用户系统
**功能描述**:
- 手机号/邮箱/社交账号登录注册
- 用户画像与偏好设置
- 身份认证与会员等级
- 个人资料管理

**用户价值**: 快速注册登录，个性化内容推荐

---

### M2: FIRE规划引擎
**功能描述**:
- 退休模拟器(参数调节+可视化曲线)
- FIRE指数追踪(当前资产/目标缺口)
- 养老跨度桥接(PensionBridge策略)
- 个性化建议报告生成

**用户价值**: 科学规划财务自由路径，量化退休目标

**核心场景**:
1. 用户输入当前年龄28岁、退休年龄60岁、当前资产42万
2. 系统计算出FIRE指数35.42%，目标缺口452万
3. 展示资产增长曲线，显示关键节点
4. 生成个性化养老规划报告

---

### M3: 社区广场
**功能描述**:
- 信息流(精选报告/FIRE实践/独身生活/城市评测)
- 内容发布(图文/视频+话题标签+地理位置)
- 互动系统(点赞/评论/收藏)
- 用户关注与消息通知

**用户价值**: 连接同路人，分享FIRE经验，获取实用建议

**内容类型**:
- FIRE实践案例
- 城市生活成本评测
- 数字游民经验分享
- 理财投资心得
- 精选报告和研究

---

### M4: 城市发现
**功能描述**:
- 城市评分卡(生活成本/医疗/空气质量/数字游民友好度)
- 城市对比工具
- 数字游民热门榜单
- 养老宜居城市推荐

**用户价值**: 找到最适合养老/生活的城市，降低生活成本

**评分维度**:
- 生活成本 (0-10分)
- 医疗水平 (0-10分)
- 空气质量 (0-10分)
- 数字游民友好度 (0-10分)
- 气候宜居 (0-10分)
- 文化包容 (0-10分)

**核心场景**:
- 对比清迈(月成本¥6,200)和上海(月成本¥18,500)
- 系统建议：选择清迈可提前3.5年退休

---

### M5: AI顾问
**功能描述**:
- 多轮对话式咨询
- 基于用户数据的个性化规划
- RAG知识库(财税/保险/投资策略)
- 报告文件生成与下载

**用户价值**: 7×24小时专业理财顾问，个性化建议

**知识库覆盖**:
- 养老金政策解读
- 税务优化策略
- 保险配置方案
- 投资组合建议
- 各国签证政策

---

### M6: 资产管理
**功能描述**:
- 总资产看板(CNY/多币种)
- 持仓明细(养老金账户/基金/保险)
- 收益追踪与风险评估
- 月度资产分析报告

**用户价值**: 一站式管理所有养老资产，实时追踪FIRE进度

**支持资产类型**:
- 个人养老金账户
- 公募基金产品
- 商业保险
- 银行存款
- 其他投资

---

### M7: 知识库
**功能描述**:
- 精选报告(城市/FIRE策略/法律合规)
- 专题内容分类
- 搜索与筛选
- 内容收藏与分享

**用户价值**: 系统学习FIRE知识，获取专业报告

**内容分类**:
- 精选报告
- FIRE实践
- 城市评测
- 法律合规

## 4. 用户旅程

### 新用户引导
1. 下载App → 引导页介绍核心价值
2. 注册登录 → 快速手机号登录
3. 填写基础信息 → 年龄、收入、资产
4. 生成第一份FIRE计划 → 查看退休模拟结果
5. 浏览社区内容 → 发现同路人经验

### 核心使用路径
1. **规划路径**: FIRE模拟 → 调整参数 → 查看建议 → 保存计划
2. **社交路径**: 浏览Feed → 点赞评论 → 发布内容 → 关注用户
3. **决策路径**: 搜索城市 → 查看评分 → 对比城市 → 咨询AI
4. **管理路径**: 添加资产 → 查看总览 → 追踪收益 → 查看报告

## 7. 产品矩阵 🆕

### 7.1 iOS App (主产品)
**定位**: C端用户移动应用  
**目标用户**: 25-40岁追求FIRE的年轻人  
**核心功能**: FIRE规划、社区社交、城市发现、AI顾问、资产管理

### 7.2 管理后台 (Admin)
**定位**: 内部运营管理系统  
**目标用户**: 产品运营、内容审核、数据分析人员  
**核心功能**:
- 数据看板: 用户数/内容数/DAU等核心指标
- 城市管理: 城市数据维护、评分更新
- 知识库管理: 文章编辑、分类管理
- 用户管理: 用户查询、会员管理、封禁操作
- 内容管理: 帖子审核、评论管理
- 系统配置: 参数设置、管理员管理

**技术栈**: Vue3 + Element Plus + ECharts  
**部署方式**: Nginx 静态部署

### 7.3 官方网站 (Portal) 🆕
**定位**: 品牌展示与用户获取  
**目标用户**: 潜在用户、媒体、投资人  
**开发时机**: Sprint 14 (App 上线后最后开发) 🔴  
**核心功能**:
- 产品介绍与功能展示
- 城市数据公开版(20个热门城市)
- 知识中心与博客
- 会员定价说明
- App 下载引导

**技术栈**: React 18 + Next.js 14 + TypeScript + GSAP 3.12+  
**部署方式**: Vercel 免费托管

**动画亮点** (Apple 级别):
- GSAP ScrollTrigger 滚动动画
- 固定容器内容切换效果
- 横向滚动画廊
- 3D 卡片倾斜交互
- 数字滚动计数动画
- 图片序列滚动播放
- 文字分裂入场动画
- 磁性按钮效果
- 平滑滚动体验

**页面结构**:
1. 首页: Hero + 功能展示 + 数据统计 + 定价 + 下载CTA
2. 产品功能页: 7大功能模块详细介绍
3. 城市数据页: 城市列表 + 评分雷达图 + 对比工具
4. 知识中心: 文章列表 + Markdown渲染
5. 定价页: 3个套餐对比 + FAQ
6. 关于我们: 团队介绍 + 愿景 + 媒体报道

**性能目标**:
- Lighthouse Performance > 95
- LCP < 2.5s, FID < 100ms, CLS < 0.1
- 首屏加载 < 2s
- 所有动画 60fps 流畅运行

**开发时长**: 11天

## 8. 竞品分析

### 直接竞品
- **理财类App**: 缺少社区和生活方式内容
- **社区类App**: 缺少专业的FIRE计算工具
- **城市评测网站**: 缺少个性化和移动端体验

### 差异化优势
1. ✅ FIRE规划 + 社区 + 城市发现三合一
2. ✅ 专注年轻人和数字游民
3. ✅ AI驱动的个性化建议
4. ✅ 丰富的可视化和交互体验
5. ✅ 完整的产品矩阵(App + 后台 + 网站)
6. ✅ Apple 级别的品牌官网体验 🆕
