# 养Young 项目总览

> **版本**: v1.1 | **更新时间**: 2026-03-15

---

## 🎯 产品定位

为年轻人打造的 FIRE(财务自由)生活方式平台，集成养老规划、社区社交、城市发现、AI理财顾问于一体。

**核心价值**: 为年轻规划，为自由而生

---

## 📱 产品矩阵

### 1. iOS App (主产品)
- **定位**: C端用户移动应用
- **技术**: Swift + SwiftUI
- **状态**: 规划中 (Sprint 1-8)

### 2. 管理后台 (Admin) 🆕
- **定位**: 内部运营管理系统
- **技术**: Vue3 + Element Plus
- **状态**: 即将开发 (Sprint 0.5)
- **功能**: 数据管理、内容审核、用户管理、数据看板

### 3. 官方网站 (Portal) 🆕
- **定位**: 品牌展示与用户获取
- **技术**: React + Next.js 14 + GSAP
- **状态**: 最后开发 (Sprint 14)
- **亮点**: Apple 级别动画效果、滚动视差、3D交互

## 🛠️ 技术栈

### 前端
- **iOS App**: Swift + SwiftUI
- **管理后台**: Vue 3 + Vite + Element Plus
- **官方网站**: React 18 + Next.js 14 + GSAP 🆕

### 后端
- **框架**: Node.js + Fastify
- **数据库**: MongoDB + Redis
- **认证**: JWT
- **AI服务**: LangChain + OpenAI/Claude (付费)

### 开发工具
- **版本控制**: Git
- **API文档**: Swagger
- **监控**: Winston + Sentry

---

## 💡 核心特性

### iOS App
1. **FIRE规划引擎**: 退休模拟器、FIRE指数计算
2. **社区广场**: 信息流、内容发布、互动系统
3. **城市发现**: 城市评分、对比工具、数字游民榜单
4. **AI顾问**: 多轮对话、个性化建议、RAG知识库
5. **资产管理**: 总资产看板、持仓明细、收益追踪
6. **知识库**: 精选报告、FIRE实践、城市评测

### 管理后台 🆕
1. **数据看板**: DAU/MAU、内容统计、趋势图表
2. **城市管理**: 城市数据 CRUD、评分维护
3. **知识库管理**: 文章编辑、Markdown支持
4. **用户管理**: 用户查询、会员管理、测试用户创建
5. **内容管理**: 帖子审核、评论管理
6. **系统配置**: 参数设置、管理员管理

## 💰 重要规则

### ⚠️ 付费操作确认规则

**核心原则**: 所有需要付费的操作都要停下来询问，确认后再继续

**付费服务清单**:
- ❌ AI API (OpenAI/Claude)
- ❌ 文件存储 (OSS/S3)
- ❌ 短信验证码
- ❌ 向量数据库付费套餐
- ❌ 监控服务付费版

**开发建议**:
- ✅ 使用管理后台维护数据
- ✅ 使用默认资源(头像/图片)
- ✅ 使用 Mock 服务
- ✅ 本地存储代替云端上传

## 📊 数据库设计

### 核心集合 (12个)

1. `users` - 用户基础信息
2. `profiles` - 用户扩展资料
3. `posts` - 社区内容
4. `comments` - 评论与回复
5. `cities` - 城市数据
6. `fire_plans` - FIRE计划
7. `assets` - 用户资产
8. `conversations` - AI对话
9. `knowledge_base` - 知识库
10. `reports` - 生成的报告
11. `notifications` - 通知
12. `admins` - 管理员 🆕
