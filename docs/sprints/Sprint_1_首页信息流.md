## 任务清单


### 功能

- [ ] Tab导航框架(首页/规划/发现/资产/我的)
- [ ] 首页Feed流(下拉刷新+上拉加载)
- [ ] 内容卡片(图文/视频渲染)
- [ ] 话题标签筛选
- [ ] 使用默认头像和种子内容数据

### 前端实现

- [ ] 完整的列表UI
- [ ] 下拉刷新/上拉加载动画
- [ ] 卡片组件(PostCard)
- [ ] 分页加载

### 后端

- [ ] GET /posts (分页查询，返回数据库数据)
- [ ] GET /posts/:id (返回帖子详情)
- [ ] POST /posts/:id/like (点赞数据入库)
- [ ] GET /tags/trending (返回热门标签)

---

## 验收标准

- [ ] Tab切换动画流畅(60fps)
- [ ] Feed流支持下拉刷新，显示加载动画
- [ ] 滚动到底部自动加载下一页(20条/页)
- [ ] 视频卡片支持封面显示（使用占位图）
- [ ] 点击话题标签筛选对应内容

---

## 测试用例

TC-1.1: 首页加载
- 打开App
- 首页显示至少10条内容（来自数据库）
- 包含图文、视频、纯文本卡片
TC-1.2: 下拉刷新
- 首页下拉
- 显示刷新动画
- 重新加载最新内容
TC-1.3: 上拉加载更多
- 滚动到列表底部
- 自动加载下20条
- 显示"加载中..."
TC-1.4: 话题筛选
- 点击"#FIRE生活"标签
- 列表仅显示包含该标签的内容
- 顶部显示筛选提示
TC-1.5: 数据持久化
- 点赞某条帖子
- 刷新页面
- 点赞状态保持
