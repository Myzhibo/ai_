## 任务清单


### 功能

- [ ] 城市列表(搜索+分类筛选)
- [ ] 城市详情卡(评分维度可视化)
- [ ] 城市对比工具(2-3城市并列)
- [ ] 推荐算法展示

### 前端实现

- [ ] 城市列表UI（卡片式）
- [ ] 城市详情页（雷达图）
- [ ] 城市对比页面
- [ ] 搜索和筛选功能

### 后端

- [ ] GET /cities (分页查询，支持筛选排序)
- [ ] GET /cities/:id (返回城市详情)

## 验收标准

- [ ] 城市列表支持按"匹配度"、"生活成本"、"医疗水平"排序
- [ ] 城市详情卡显示6个维度评分(雷达图)
- [ ] 城市对比支持2-3个城市并列对比
- [ ] 推荐算法基于用户数据

## 测试用例

TC-5.1: 城市搜索
- 输入"清迈"
- 从数据库查询
- 显示"清迈·泰国"卡片
TC-5.2: 城市详情
- 打开清迈详情
- 显示6个维度评分雷达图
- 数据来自数据库
TC-5.3: 城市对比
- 选择上海、清迈
- 并列显示对比卡
- 对比项: 生活成本¥18,500 vs ¥6,200
- 显示"提前3.5年退休"建议
TC-5.4: 智能推荐
- 显示推荐城市列表
- 基于用户FIRE目标推荐
- 每个城市显示推荐理由
TC-5.5: 筛选功能
- 筛选"生活成本<¥10,000"
- 显示符合条件的城市
- 按匹配度排序
