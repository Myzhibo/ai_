# Sprint 0: 基础设施

**时长**: 2周  
**状态**: 📋 待开始  
**优先级**: P0 (最高)

---

## 目标

搭建项目基础框架，建立前后端开发环境，完成数据库设计和种子数据导入。

---

## 任务清单

### 前端
- [ ] 项目脚手架 + SwiftUI Design System
- [ ] 网络层封装(APIClient + Interceptor)
- [ ] 路由管理(Coordinator Pattern)
- [ ] 本地存储模块(Core Data Models)

### 后端
- [ ] Fastify服务初始化 + 中间件配置
- [ ] MongoDB连接池 + Redis配置
- [ ] JWT认证中间件
- [ ] 统一响应格式 + 错误处理
- [ ] API接口文档生成器

### 数据库
- [ ] 所有集合Schema定义
- [ ] 索引策略配置
- [ ] 数据迁移脚本
- [ ] 种子数据导入(城市/知识库/测试用户等)

---

## 验收标准

- [ ] Design System能独立预览所有组件(Button/Input/Card等)
- [ ] API请求可正常发送并处理401/500错误
- [ ] MongoDB所有集合Schema已定义并建立索引
- [ ] 种子数据可一键导入(至少50个城市数据)
- [ ] Swagger文档可访问并测试接口
