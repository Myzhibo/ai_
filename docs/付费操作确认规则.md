## 💡 开发建议

### 开发阶段（推荐）
1. ✅ **使用 MongoDB 种子数据**：预先准备测试数据，无需付费服务
2. ✅ **使用默认资源**：头像、图片等使用本地默认资源
3. ✅ **使用 Mock 服务**：模拟第三方 API 响应
4. ✅ **本地存储**：文件暂存本地，不上传云端
5. ✅ **开源替代方案**：优先使用开源免费服务

### 测试阶段（需确认）
- 🔶 使用免费额度的服务（需提前说明）
- 🔶 使用测试环境的付费服务（需明确成本）
- 🔶 小规模试用付费功能（需预算审批）

### 生产阶段（必须确认）
- 🔴 所有付费服务必须获得书面批准
- 🔴 必须明确月度/年度预算
- 🔴 必须有成本监控和告警机制

## 📝 使用流程

### 发现需要付费服务时

```
1. 🛑 立即停止开发
   ↓
2. 📋 记录需要的服务、功能、原因
   ↓
3. 💰 评估成本（月度/年度）
   ↓
4. 🔍 寻找免费替代方案
```
